# Prins Willem Alexander Manege

Voor deze Performance audit heb ik de website van de [Prins Willem Alexander Manege](https://www.pwamanege.nl/) getest. De Prins Willem Alexander Manege is onderdeel van de 'Vereniging Paardrijden Gehandicapten', wat inhoud dat de manege rekening houd met mensen met verschillende handicaps zodat iedereen kan paardrijden. 
<br> Op deze website krijg je meer informatie over de manege en hoe je je kunt aanmelden om daar te kunnen paardrijden.

## Samenvatting testbevindingen 
De website presteert beter op desktop dan op mobiel, vooral doordat er dan minder ongebruikte JavaScript is en de 'main-thread work' is geminimaliseerd. Op mobiel kunnen de prestaties verbeterd worden door ongebruikte JavaScript en CSS te minimaliseren/ verwijderen, afbeeldingen te comprimeren en minder Fonts in te laden. 

Voor de volledige documentatie van mijn onderzoek: [zie de wiki](https://github.com/SieuwkeSheta/performance-audit/wiki/Performance-Audit-Test-%E2%80%90-Prins-Willem-Alexander-Manege).
